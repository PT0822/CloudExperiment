## 2. Kubernetes 资源清理命令

在 CloudShell 或配置好 kubeconfig 的终端执行：

```bash
# 第一部分 Web 应用资源
kubectl delete -f k8s/ --ignore-not-found

# Spark 作业和性能对比 Job
kubectl delete sparkapplication wordcount-demo --ignore-not-found
kubectl delete job spark-douban-analysis spark-perf-1 spark-perf-2 --ignore-not-found

# MQTT 附加题资源
kubectl delete -f addons/edge-mqtt/cloud/ --ignore-not-found
kubectl delete job edge-mqtt-publisher --ignore-not-found

# 显式清理可能产生 ELB 的 Service
kubectl delete svc frontend-svc backend-lb-svc backend-svc redis-svc mqtt-broker --ignore-not-found

# 监控和 Spark Operator
helm uninstall kps -n monitoring || true
helm uninstall spark-operator -n spark-operator || true
kubectl delete ns monitoring spark-operator --ignore-not-found

# 最后清理 PVC；会删除持久化数据，不可恢复
kubectl delete pvc redis-data-pvc spark-data-pvc --ignore-not-found
```

## 3. 华为云控制台清理

| 资源 | 控制台位置 | 操作 | 清理后截图 |
|---|---|---|---|
| CCE 集群/节点 | 云容器引擎 CCE | 删除集群，或删除节点池/节点 | 集群已删除，或节点数为 0 |
| ELB | 弹性负载均衡 ELB | 删除由 LoadBalancer Service 创建的监听器、后端服务器组、负载均衡实例 | 无相关 frontend/backend ELB |
| EIP | 弹性公网 IP | 释放实验公网 IP | 无绑定实验资源 |
| EVS 云硬盘 | 云硬盘 EVS | 删除 PVC 遗留磁盘；检查快照 | 无 redis-data-pvc/spark-data-pvc 对应磁盘 |
| OBS 桶 | 对象存储 OBS | 删除数据集对象；如不再需要则删除桶 | 桶为空或已删除 |
| SWR 镜像 | 容器镜像服务 SWR | 建议评分结束后再删镜像和组织 | 镜像保留或删除结果 |
| 费用中心 | 费用中心 | 检查活跃资源和近期扣费 | 无异常持续扣费 |
